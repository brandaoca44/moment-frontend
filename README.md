# 🌐 Moment Web (Frontend)

Interface web da rede social **Moment**, focada em experiência simples, rápida e moderna.

---

## 🧠 Sobre o projeto

O frontend do Moment entrega uma experiência leve e fluida, com foco em:

* Feed infinito
* Interações rápidas (Loved e ReMont)
* Perfil de usuário
* Autenticação segura com cookies
* UI limpa e moderna

---

## ⚛️ Tecnologias

* React
* TypeScript
* Vite
* React Query
* Axios
* Tailwind CSS

---

## ⚙️ Configuração

### 1. Clone o projeto

```
git clone https://github.com/seu-user/moment-frontend.git
cd moment-frontend
```

### 2. Instale as dependências

```
npm install
```

### 3. Configure o .env

```
VITE_API_URL=http://localhost:3000
```

---

## ▶️ Rodando o projeto

```
npm run dev
```

App:

```
http://localhost:5173
```

---

## 🔗 Integração com backend

O frontend consome a API do backend via:

* Cookies HTTP-only (auth segura)
* React Query (cache e sincronização)
* Axios (requisições)

---

## 💡 Funcionalidades

* Login / Cadastro
* Feed com scroll infinito
* Criar post
* Curtir (Loved)
* Repost (ReMont)
* Seguir usuários
* Bloquear usuários
* Perfil
